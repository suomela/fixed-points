# Problem

    R X X
    B Y Y

    X X
    Y Y
    R BY

# Intuition
 
Nodes of the 3-regular tree are either red (R X X) or blue (B Y Y). Nodes of the same color form paths: red paths are infinite while blue paths may be finite.
